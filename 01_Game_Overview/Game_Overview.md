## 2. 핵심 개요

> **시놉시스 요약**  
> 비밀 실험 *BLEACH* 로 탄생한 “색을 잃은 신인류”가 도시 *이리시아*를 장악.  
> 바이러스 면역자 3인은 각 구역을 지배하는 적성자(보스)들을 쓰러뜨려  
> 도시의 색과 생명을 되찾아야 한다.

| 구분 | 내용 |
| --- | --- |
| 플레이 구조 | 3인 팀 실시간 스위치 + 보스 헌팅(5구역 클리어) |
| 게임 모드 | **스토리**(선형 진행) · (추후) 로그라이트 모드 |
| 난이도 | 쉬움 / 보통 / 어려움 (피해량 ±10 %) |
| 평균 플레이타임 | 베타 1회차 약 3 시간 |

## 3. 타깃 오디언스

| 세그먼트 | 특징 | NOVA 제공 포인트 |
| --- | --- | --- |
| 핵 앤 슬래시 팬 | 호쾌한 범위 공격·파밍 | 3인 콤보 + 픽토스 빌드 |
| 좀비·포스트 아포칼립스 팬 | 바이러스·돌연변이 서사 | 흑백 x 선홍 비주얼, 돌연변이 보스 |
| 라이트 액션 입문자 | 간단 입력, 명확 난이도 | 4버튼 조작·친절 튜토리얼 |

## 4. 콘텐츠 스냅숏 (베타 범위)

| 챕터 | 무대 | 메인 목표 | 보스 |
| --- | --- | --- | --- |
| 프롤로그 | 면역자 수용소 → 슬럼 | 탈출 & 항체 샘플 확보 | – |
| S-1 | 하이브 타워 | 데이터 서버 해킹 | – |
| S-2 | 로우라인 지하 | 적성자 ‘말머’ 추격 | **말머** |
| S-3 | 레드 스트립 | 적성자 ‘살럼’ 격파 & 떡밥 | **살럼** |

## 5. 시스템 한눈에

| 카테고리 | 요약 |
| --- | --- |
| **조작** | 약공(LMB) · 강공/패링(RMB) · 회피(Space) · 스위치(Q/E) |
| **팀 스위치** | 교대 시 캐릭터 고유 버프, 무적 0.3 s |
| **픽토스** | 8 슬롯 / 20 코스트, 레벨 0-3, 고코스트 시 부작용 |
