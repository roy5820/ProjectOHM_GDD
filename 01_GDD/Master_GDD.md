## 2. 세계관 ― “안개 속으로 들어갈 자격”

### 2-1. 시기 & 사건  
1999년 11월, 각국 도시에 정체불명의 안개 기둥이 솟아오르며 **‘리프트’** 가 열린다. 내부는 네온 정글·수중 공중도시 같은 판타지 생태계로 치환되고, 시공률이 뒤틀려 **외부 1 분 = 내부 1 시간**이 된다.

### 2-2. 진입 기술 **〈워프-앵커 장치〉**  
| 규칙 | 설명 |
|------|------|
| 안전화 | 일반인은 즉시 분해 → 앵커 탑재 요원만 생존 가능 |
| 파티당 갯수 | 앵커 **3개** 한정 |
| 전투 방식 | 필드에 **1명**만 존재, 교대 시 워프-인·아웃 |
| 교대 트리거 | `Shift`(전) / `Space`(후) 입력 |

### 2-3. 대립 구도  
| 세력 | 목적 |
|------|------|
| **R-SEC** | 리프트 봉쇄, 민간 구조 _(플레이어)_ |
| **피스톤 재단** | 에테르 무장화·판매 |
| **이계 군주 ‘아르카나’** | 지구를 리프트화·식민지화 |

## 3. 핵심 게임 플레이 루프
1. **기지** : 파티 편성 · 특성 강화 · 캐릭터 해금  
2. **리프트** : 절차 생성 10-14룸(전투·이벤트·보급)  
3. **보스 ‘Heart of Rift’** 처치 → 60 초 붕괴 타이머 → 탈출  
4. **결산** :  
   - 인던 통화 `에테르 코어` 사용 내역  
   - 메타 통화 `아카이브 크레딧` 획득 → 특성·해금

## 4. 전투 시스템

### 4-1. 공통 파라미터
| 리소스 | 설명 |
|--------|------|
| **HP** | 캐릭터별 고유, 물약으로만 회복 |
| **SP** | 파티 공유 0-10 (Relic으로 15까지), 평타 적중 시 충전 |

### 4-2. 캐릭터 교체 규칙 (워프 스위칭)
1. 입력 `Shift` / `Space`  
2. 교체 순간 0.3 s 무적 + **0.5 s 재스위치 불가**  
3. 교체 전 요원은 **현재 액션 완수 후** 페이드-아웃  
4. 새 요원 등장 시 SP 소모 **체인 아츠** 시작

### 4-3. 일반공격 규칙  
* 캐릭터마다 **콤보 타수·모션·SP 수급량이 상이**  
* 콤보는 회피·스위치로만 캔슬 가능  

### 4-4. 회피 대시 규칙
| 입력 | 무적 | 쿨 |
|------|------|----|
| `Q`  | 0.25 s | 0.8 s |

### 4-5. 체인 아츠 사용 규칙
* 교체 순간 SP 1/3/5 소비, 캐릭터별 3단계  
* 발동 중 교체 불가 → SP 소진 후 0.5 s부터 가능  

### 4-6. 물약 사용 규칙
| 스택 | 효과 | 캔슬 |
|------|------|------|
| 파티 공유 3 | 5 s 동안 HP 20 % 회복 | 회피·교체로 **불가** |

## 5. 장비 시스템

### 5-1. 유물 장착 규칙
| 항목 | 내용 |
|------|------|
| 획득 | 상호작용 → **3개 중 1개 선택** |
| 중복 방지 | 소유 유물은 목록에 미출현 |
| 제한 | 장착 한도 없음 · 버프형과 리스크형 혼재 |
| 리롤 | `에테르 코어` 소모, 시도마다 비용 2× 증가 |

> **예시**  
> - `수평선의 성배` : 교체 시 파티 HP + 2 %  
> - `시간 파열석` : 회피 쿨 −30 %, 이동속도 −15 %

### 5-2. 룬 장착 규칙
| 항목 | 규칙 |
|------|------|
| 획득 | 필드 드랍 → `E` 상호작용 시 3캐릭터 슬롯 UI |
| 슬롯 | 캐릭터당 3칸 |
| 중복 | **동일 룬은 파티 내 1개만** |
| 장착 | 캐릭터 초상 클릭 → 장착 / 가득 차면 룬 분리 → 필드 드랍 |
| 강화 | 상점에서 `에테르 코어` 사용 (Lv1-3) |

### 5-3. 인던 통화 **`에테르 코어`**
| 용도 | 설명 |
|------|------|
| 유물 리롤 | 1→2→4→8… 배수로 비용 증가 |
| 룬 구매·강화 | 상점 전용 |
| 물약 충전 | 1개당 고정가 |
