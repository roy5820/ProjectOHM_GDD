## 2. ⚔ 전투 시스템

### 2-1. 전투 방식

- 슈팅 기반 실시간 액션
- 3인 고정 파티 (캐릭터 교체 없음)
- 일반공격: 자동 조준 투사체, 장탄 수 존재, 자동 또는 R키 수동 장전, 제한 없음
- 회피, 액티브 스킬, 패시브 스킬, 궁극기로 구성
- 충격력 시스템: 적에게 누적하여 그로기 유발
- 궁극기 에너지 시스템: 일반공격/스킬로 충전

### 2-2. 캐릭터 공통 전투 스탯

| 스탯 이름     | 코드명     | 설명 |
|--------------|------------|------|
| 체력          | HP         | 캐릭터 생명력 |
| 공격력        | ATK        | 데미지 계산 시 계수에 곱해짐 |
| 방어력        | DEF        | 받는 피해 감소 |
| 이동 속도     | SPD        | 기본 이동 속도 배율 |
| 치명타 확률   | CRIT       | 치명타 발생 확률 |
| 치명타 배수   | CRIT_DMG   | 치명타 발생 시 배수 적용 |
| 충격력        | STG        | 타격 시 그로기 게이지에 누적되는 계수 |
| 궁극기 효율   | ULT_EFF    | 궁극기 에너지 획득량 배율 |

### 2-3. 스킬 구성

| 스킬 종류     | 설명 |
|--------------|------|
| 일반공격      | 자동 조준 투사체 / 장탄 수 존재 / 장전 필요 / 투사체 수치 통일 |
| 액티브 스킬   | 쿨타임 기반의 고유 스킬 |
| 패시브 스킬   | 조건부 또는 지속 효과 제공 |
| 궁극기        | 에너지 축적 후 발동되는 강력한 기술 |
| 회피          | 무적 프레임 포함, 방향 기반 이동기 |

### 2-4. 수치 계산 공식

- **데미지** = 데미지 계수 × ATK × (CRIT_DMG 적용 여부)  
- **충격력** = 충격력 계수 × STG  
- **궁극기 에너지** = 획득량 × ULT_EFF

### 2-5. 회피 (공용 시스템)

| 항목              | 수치 / 설명 |
|-------------------|-------------|
| 무적 지속 시간     | 약 0.35초 |
| 회피 거리          | 약 3~4m |
| 쿨타임             | 2.0초 |
| 입력 방식          | 방향키 + 회피 키 (예: Shift) |
| 회피 성공 기준     | 회피 중 무적 프레임으로 적의 공격을 회피한 경우 |
| 저스트 회피 조건   | 회피 발동 후 0.2초 이내에 적의 공격을 무적 프레임으로 회피 |
| 저스트 회피 효과   | 궁극기 에너지 +1 / 2초간 무적 상태 추가 부여 (전 캐릭터 공통 적용) |

## 3. 🧑 캐릭터 예시 – 레오

### 3-1. 기본 정보

- 타입: 밸런스형 전사

- 특징: 빠른 연속 사격 + 근거리 스킬

## 6. 🧾 모드(Mod) 시스템

### 6-1. 시스템 개요

| 항목             | 내용 |
|------------------|------|
| 시스템 명칭       | 모드(Mod) |
| 컨셉             | 워프레임의 모딩 시스템에서 착안 |
| 획득 방식        | 게임 중 드랍 (중복 드랍 없음) |
| 강화 시스템      | 없음 |
| 상위 버전 존재   | 있음 (일부 모드는 Lv2, Lv3까지 존재) |
| 중복 장착 제한   | 동일 캐릭터 내 중복 장착 불가 |
| 공유 장착 허용   | 다른 캐릭터 간 동일 모드 사용 가능 |
| 최대 장착 슬롯   | 8칸 |
| 최대 장착 용량   | 20 |

### 6-2. 장착 규칙

| 규칙 항목                   | 설명 |
|----------------------------|------|
| 장착 슬롯 수               | 캐릭터당 최대 8개 |
| 최대 용량                  | 총합 20까지 장착 가능 |
| 동일 모드 중복 장착        | ❌ 하나의 캐릭터 안에서는 불가 |
| 다른 캐릭터 간 공유        | ✅ 동일 모드를 서로 사용할 수 있음 |
| 상위 버전 단계             | Lv1 / Lv2 / Lv3 또는 None |
| 강화 여부                  | 강화 기능 없음 (상위 버전은 별도 획득) |

### 6-3. 모드 레벨 규칙

| 레벨   | 설명 |
|--------|------|
| None   | 고정형 모드, 레벨 없음, 효과와 코스트 고정 |
| Lv1    | 시작 단계, 가장 낮은 성능 |
| Lv2    | 중간 단계, 효과 및 용량 상승 |
| Lv3    | 최고 단계, 가장 높은 성능 및 용량 |

### 6-4. 모드 예시

| 모드 이름     | 레벨 | 용량 | 효과 |
|----------------|------|------|-------------------------------|
| 아드레날린     | Lv1  | 5    | 체력 30% 이하일 때 공격력 +10% |
| 아드레날린     | Lv2  | 7    | 체력 30% 이하일 때 공격력 +18% |
| 아드레날린     | Lv3  | 9    | 체력 30% 이하일 때 공격력 +27% |
| 버서커         | Lv1  | 6    | 일반공격 속도 +15%             |
| 버서커         | Lv2  | 8    | 일반공격 속도 +25%             |
| 강인한 정신    | None | 4    | 피격 시 경직 감소              |
| 약점 노출      | None | 8    | 치명타 확률 +10%               |

- 사용 용량: 18 / 20

  - 아드레날린 Lv2 (7)
